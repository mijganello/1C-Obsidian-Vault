В платформе 1с существуют *типы данных*.

*Типы данных* делятся на два типа:
1. Примитивные
2. Типы, образованные Типообразующими Объектами конфигурации

*Примитивные типы*:
1. Число
2. Строка
3. Дата
4. Булево

Объекты конфигурации, которые могут образовывать новые типы данных, называются *типообразующими*.

*Типообразующие Объекты конфигурации*, например: [[Справочник]], при создании автоматически создают некоторые типы данных в системе.

Примеры таких типов для *Справочника Номенклатура*:
1. СправочникМенеджер.Номенклатура
2. СправочникСсылка.Номенклатура
3. СправочникОбъект.Номенклатура
4. СправочникВыборка.Номенклатура

```
Следует еще раз отметить, что эти типы данных не поддерживаются платформой изначально и существуют только в конкретном прикладном решении.
```

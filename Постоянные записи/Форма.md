*Форма* - объект конфигурации, отвечающий за визуальное представление интерфейса заполнения некоторых данных в пользовательском интерфейсе.

Формы бывают следующих *типов*:
1. *Форма объекта* (элемента справочника, документа) - Предназначена для создания и редактирования данных конкретного объекта.
2. *Форма списка* - Отображает список записей, поддерживает сортировку, поиск и фильтрацию.
3. *Форма выбора* - Используется для поиска и выбора значения из списка (например, выбор номенклатуры в документе).
4. *Форма группы* (для иерархических справочников) - Специализированная форма для редактирования групп справочника.
5. *Произвольная форма* - Создается разработчиком вручную, не привязана к конкретному объекту метаданных (например, форма обработки).

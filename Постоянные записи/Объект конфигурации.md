### Что такое Объект конфигурации

**Объекты конфигурации** – логические единицы, «блоки», из которых
состоит конфигурация.

Примеры объектов конфигурации:
1. [[Подсистема]]
2. [[Справочник]]
3. [[Документ]]
4. [[Регистр накопления]]

У каждого объекта конфигурации существует различный набор *свойств*, который определяется в основном назначением.

Примеры свойств объекта конфигурации:
1. Имя
2. Синоним
3. Комментарий
4. Тип данных
5. и т.д.

**Объекты конфигурации** могут быть:
1. *Прикладными* - самостоятельные, отражают какую-то часть бизнес-логики.
2. *Подчиненные* - объекты конфигурации, существующие только в рамках другого объекта. Пример: *реквизит*.

**Окно редактирования** свойств объекта конфигурации предназначено
для сложных объектов конфигурации и позволяет путем выполнения
последовательных действий быстро создавать такие объекты.

*Свойства* объектов также могут быть отредактированы в *окне свойств*.

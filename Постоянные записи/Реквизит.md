### Что такое Реквизит

*Реквизит* - подчиненный объект конфигурации, предназначенный для описания набора дополнительной информации для некоторого самостоятельного объекта конфигурации. 

Описанная реквизитами информация будет *одинаковой для каждого экземпляра самостоятельного объекта конфигурации*, для которого создаются реквизиты.

*Реквизит*, как и любой [[Объект конфигурации]] имеет набор свойств, среди основных:
1. Имя
2. Синоним
3. Тип
4. Значение заполнения
5. Проверка заполнения

Объекты конфигурации, для которых возможно создание реквизита имеют набор *стандартных реквизитов*, различных для каждого объекта конфигурации, например:
1. Код
2. Наименование

Доступность стандартных реквизитов зависит также от свойств объекта конфигурации для которого он создаётся. Например при включённом свойстве *иерархический* у [[Справочник|справочника]], у его элемента становятся доступен стандартный реквизит *Родитель*.
